Usage:

Pass shadowJs a target element that already contains a tree of elements to clone into the shadow. This allows you to load additional libraries that manipulate the whole document without changing other elements.

You can add a listener with onUpdateShadow to run your code every time you call or add a function inside the shadow. 

<div id="shadow-container">
  <p>text to change</p>
</div>
